Exhibition Mod — Player install

Quick start

  1. Unzip this mod into the same folder as your game HTML.
  2. Make sure your game file is named CourseOfTemptationtest.html (or leave the default CourseOfTemptation.html — the installer will find it).
  3. Run the installer:
    • Linux: ./install.sh
    • Windows: double-click install.exe (or install.bat if Python is installed)
  4. Open your game HTML and hard-refresh (Ctrl+Shift+R).
  5. Open the Paperdoll asset editor via Exhibition Paperdoll Editor.sh (Linux) or Exhibition Paperdoll Editor.bat (Windows).

Uninstall (one click + Yes/No popups)

  1. Windows: double-click uninstall.bat
  2. Linux: double-click Uninstall Exhibition Mod.desktop or uninstall.sh

No command line needed. Two popups appear:

  1. Uninstall Exhibition Mod? — Yes continues, No cancels
  2. Remove mod folders too? — Yes = full clean; No = only restore the game HTML (keep files for later)

What the installer does

  • Backs up your game HTML
  • Rebakes every Exhibition mod feature into your game file
  • Rebuilds the standalone Paperdoll editor so it works offline
  • Creates desktop-style launcher scripts for the editor
  • Writes backup/install-record.json so uninstall can restore cleanly

Mod authors

While developing, use ./rebake-dev.sh on your dev HTML copy.

Before publishing the ZIP, run build-install-exe.bat on Windows once to create install.exe for players without Python.
